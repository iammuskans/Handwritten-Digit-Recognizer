# 🧠 AI Handwritten Digit Recognizer

An AI-powered web app that recognizes handwritten digits (0–9) using a Convolutional Neural Network (CNN) trained on the MNIST dataset.

## 🚀 Features

- Real-time digit recognition
- CNN model with ~99% accuracy
- Interactive drawing canvas
- AI confidence gauge
- Top-3 digit predictions
- Prediction history dashboard
- Modern glassmorphism UI

## 🛠 Tech Stack

Python  
TensorFlow / Keras  
Streamlit  
Plotly  
NumPy  
Pillow  

## 📊 Model

The model is a CNN trained on the MNIST dataset.

Architecture:
- Conv2D
- MaxPooling
- Dense layers
- Softmax output (10 classes)
